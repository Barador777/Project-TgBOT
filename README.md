# Бот - напоминалка

<img width="1159" height="572" alt="{FC65FE36-F5E8-49E6-A956-9DB77C7579DE}" src="https://github.com/user-attachments/assets/8e692263-04c2-4c62-8934-422db062e31a" />

Этот бот поможет вам эффективно управлять временем и не забывать важные дела. Просто создайте напоминание и бот вовремя напомнит о встречах, задачах, днях рождения или других событиях. Удобный и простой интерфейс, поддержка различных форматов ввода времени и даты.

## Установка библиотек
Установка происходит через командную строку ОС. У вас должны быть установленны следующие библиотеки:
1. pyTelegramBotAPI 
  ```pip install pyTelegramBotAPI```

2. apscheduler 
  ```pip install apscheduler```

3. sqlite3 
(Встроена в Python по умолчанию)

## За что отвечает каждая библиотека
| Библиотека       | Зачем нужна                                                                             |
|------------------|-----------------------------------------------------------------------------------------|
| pyTelegramBotAPI | База для создания Telegram-ботов на языке Python                                        |
| apscheduler      | С её помощью можно запускать задания в определённые моменты                             |
| sqlite3          | База данных, где бот хранит текст напоминаний, чтобы не забыть их после перезагрузки    |

Эта программа зависит от интепретатора Python версии 3.8, PIP 22.0 (или выше). 

## Плюсы бота
- Не нужно устанавливать отдельные приложения - напоминания прямо в привычном Telegram;
- Помогает повысить продуктивность и снизить стресс, связанный с забывчивостью;
- Позволяет планировать дела на день, неделю, месяц и даже года вперед;
- Идеально подходит для личного и рабочего использования.
Попробуйте этого бота прямо сейчас и начните контролировать свое время легко и эффективно!
